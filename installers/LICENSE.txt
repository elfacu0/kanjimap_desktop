This app was created to give a visual map of the connections between kanjis and to help Japanese learners study more efficiently.
The kanji decomposition is represented by a graph with detailed kanji and radical information.
If you would like to help improving this app, make sure to support it.

If you would like to report a bug, request features or just say thanks you can contact me at thekanjimap@gmail.com.
